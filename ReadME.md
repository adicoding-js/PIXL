## How to Run
- Clone the repo:
```bash
git clone [https://github.com/adicoding-js/pixl.git](https://github.com/adicoding-js/pixl.git)
```
- Install Packages
```bash
npm install
```
- Run the Node.js Server
```bash
node server.js
```
- NOTE: The app runs on port 3000 by default. Go to `http://localhost:3000` in your browser.

### OR Just play it directly on -> [PIXL](https://pixl-production.up.railway.app/) (updates every commit)

## How to Play!
- Enter your username when prompted.
- Select your canvas from the dropdown (Main, Dark, or Retro).
- Pick your tools from the toolbar (Pencil, Eraser, Eyedropper, Line, Rectangle, Circle) and select a color from the classic CGA palette!
- Draw on the canvas and watch it sync live with everyone else. You can also chat with other artists in the draggable chat window!
---
## What's in it so far
- Classic Windows 98 aesthetic (draggable windows, classic menus, outset/inset borders)
- Online Multiplayer drawing with live sync via Socket.io
- Multiple Canvas Themes (Classic, Dark, Retro)
- Advanced Shape Tools (Bresenham line, rectangle, circle with live previews before placing)
- Live chat room with draggable UI
- Interactive Minimap with panning and zoom to cursor support
- A CRT scanline effect (that kinda works)
- Live Undo functionality (Ctrl+Z) and Save Canvas option

## Known Issues
- UI windows might overlap and become clunky on smaller mobile screens, however i tried to optimize it but the game dosen't work properly.
- No permanent user authentication (usernames are claimable per session)
- Erasing with the eraser tool shows a slightly offset cursor depending on zoom levels
